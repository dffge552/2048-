📖 專案簡介 / About
中文
這是一款強化版 2048 遊戲，在經典玩法基礎上加入了多項創新機制，讓遊戲不再止步於 2048，而是可以持續挑戰極限，目標直指 2¹⁰⁰（約 1.27 溝）！
由一名熱愛程式的高中生開發，旨在探索 2048 遊戲的無限可能。

English
This is an enhanced version of the classic 2048 game, featuring multiple innovative mechanics that push beyond the original 2048 goal. The ultimate target is 2¹⁰⁰ (approximately 1.27 octillion)!
Developed by a high school student passionate about coding, this project aims to explore the infinite possibilities of the 2048 game.

✨ 特色功能 / Features
類別	功能	中文說明	English Description

智慧平衡	自動調整遊戲節奏，避免過早卡死	Auto-adjusts game pace to prevent early deadlocks
無限目標	沒有終點，目標是達到 2¹⁰⁰ 或更高	No final limit – aim for 2¹⁰⁰ or beyond
🎮 操作輔助	方向鍵 + 觸控	支援鍵盤與觸控螢幕操作	Supports both keyboard and touch input
↶ 悔步	可連續撤銷，降低失誤成本	Continuous undo to reduce costly mistakes
🧠 AI 分析	評估每個方向的分數，並給出策略建議	Evaluates each direction and provides strategic advice
📜 移動記錄	查看完整的操作歷史	View your complete move history
♾️ 無盡模式	⏱️ 限時模式	10 分鐘內盡可能累積高分	Score as high as possible within 10 minutes
♾️ 無限模式	無時間限制，挑戰極限分數	No time limit – push your score to the limit
🤖 AI 無盡	觀看 AI 自動遊玩，學習最佳策略	Watch AI play automatically and learn optimal strategies
🏆 紀錄系統	本地儲存	自動儲存所有模式的最高分	Automatically saves high scores for all modes via localStorage
